#  Bienes Raíces - Node + Express + Tailwind

Practice web project for real estate developed with Node.js, Express, MySQL, Sequelize, Pug, and Tailwind CSS.
Includes authentication system, CSRF protection, and a property management admin panel.

---

##Technologies Used
Node.js
Express
MySQL
Sequelize ORM
Pug (Template Engine)
Tailwind CSS
PostCSS
CSRF Protection
JSON Web Tokens (JWT)
